# 驾驶陪练系统教练手册

## 一、系统概述
欢迎使用驾驶陪练系统！本系统旨在帮助教练管理陪练课程，提高教学效率。您可以通过本系统设置可用时间、管理学员预约、查看课程记录等。

## 二、注册与登录

### 2.1 注册账号
1. 打开驾驶陪练系统教练端页面
2. 点击“注册”按钮
3. 填写注册信息：
   - 姓名：请输入您的真实姓名
   - 手机号：请输入您的常用手机号
   - 邮箱：请输入您的常用邮箱（可选）
   - 密码：请设置6位以上的密码
   - 区域：请选择您所在的区域
4. 点击“注册”按钮完成注册
5. 注册成功后，您的状态将变为“待确认”，等待管理员审核

### 2.2 登录系统
1. 打开驾驶陪练系统教练端页面
2. 输入您的手机号和密码
3. 点击“登录”按钮
4. 登录成功后，您将进入教练首页

### 2.3 忘记密码
1. 在登录页面点击“忘记密码”
2. 输入您的手机号
3. 点击“获取验证码”
4. 输入收到的验证码
5. 设置新密码
6. 点击“确认”按钮完成密码重置

## 三、排课管理

### 3.1 添加排课
1. 登录系统后，进入教练首页
2. 点击“添加排课”按钮
3. 填写排课信息：
   - 日期：选择您可以提供陪练服务的日期
   - 时段：选择上午、下午或晚上
   - 价格：设置该时段的陪练价格
   - 备注：填写其他需要说明的信息（可选）
4. 点击“保存”按钮
5. 排课成功后，学员可以看到您的可用时间并进行预约

### 3.2 查看排课
1. 点击底部导航栏的“排课管理”
2. 您可以看到所有已添加的排课
3. 排课状态说明：
   - 可用：未被学员预约的排课
   - 待确认：已有学员预约，等待您确认
   - 已确认：您已确认学员的预约
   - 已取消：学员已取消预约
   - 已完成：课程已结束

### 3.3 编辑排课
1. 在排课列表中找到您想要编辑的排课
2. 点击“编辑”按钮
3. 修改排课信息
4. 点击“保存”按钮
5. 注意：如果排课已有学员预约，编辑可能会影响学员的预约

### 3.4 删除排课
1. 在排课列表中找到您想要删除的排课
2. 点击“删除”按钮
3. 确认删除操作
4. 注意：如果排课已有学员预约，删除前请先通知学员

## 四、课程管理

### 4.1 查看课程列表
1. 点击底部导航栏的“我的课程”
2. 您可以看到所有已预约的课程
3. 课程状态说明：
   - 待确认：学员已预约，等待您确认
   - 已确认：您已确认学员的预约
   - 已取消：学员已取消预约
   - 已完成：课程已结束

### 4.2 确认课程
1. 在课程列表中找到“待确认”的课程
2. 点击“确认”按钮
3. 确认课程信息
4. 点击“确认”按钮
5. 确认成功后，课程状态将变为“已确认”

### 4.3 标记课程完成
1. 在课程列表中找到“已确认”的课程
2. 点击“标记完成”按钮
3. 确认课程已结束
4. 点击“确认”按钮
5. 标记成功后，课程状态将变为“已完成”

### 4.4 查看课程详情
1. 点击课程卡片进入课程详情页面
2. 查看课程的详细信息：
   - 学员姓名和联系方式
   - 课程日期和时段
   - 课程状态
   - 课程地点

## 五、学员管理

### 5.1 查看学员列表
1. 点击底部导航栏的“学员管理”
2. 您可以看到所有预约过您课程的学员
3. 查看学员的基本信息：
   - 姓名
   - 手机号
   - 所在区域
   - 预约记录

### 5.2 联系学员
1. 在学员列表中找到您想要联系的学员
2. 点击“联系”按钮
3. 选择联系方式：
   - 电话联系
   - 短信联系
   - 系统消息

### 5.3 评价学员
1. 课程结束后，您可以对学员进行评价
2. 点击课程详情页面的“评价学员”按钮
3. 对学员的学习态度、驾驶技能等进行评分和评价
4. 点击“提交”按钮

## 六、个人中心

### 6.1 个人信息
1. 点击底部导航栏的“我的”
2. 查看您的个人信息：
   - 姓名
   - 手机号
   - 邮箱
   - 所在区域
   - 教练状态

### 6.2 修改个人信息
1. 在个人中心页面点击“修改信息”
2. 修改您的个人信息
3. 点击“保存”按钮

### 6.3 修改密码
1. 在个人中心页面点击“修改密码”
2. 输入原密码
3. 输入新密码
4. 确认新密码
5. 点击“保存”按钮

### 6.4 查看收入
1. 在个人中心页面点击“收入统计”
2. 查看您的收入情况：
   - 总收入
   - 月度收入
   - 课程收入明细

### 6.5 退出登录
1. 在个人中心页面点击“退出登录”
2. 确认退出操作
3. 系统将返回登录页面

## 七、常见问题

### 7.1 如何提高课程预约率？
- 完善个人资料和教学经验
- 设置合理的价格
- 保持良好的教学态度
- 及时确认学员的预约

### 7.2 学员取消预约怎么办？
- 系统会自动通知您学员取消预约
- 您可以重新设置该时段为可用
- 建议与学员沟通取消原因，提高服务质量

### 7.3 如何处理学员的投诉？
- 认真听取学员的意见和建议
- 积极与学员沟通解决问题
- 如有需要，可以联系管理员协助处理

### 7.4 如何查看学员的评价？
- 在个人中心页面点击“我的评价”
- 查看学员对您的评价和评分
- 根据评价改进教学方法

### 7.5 如何申请提现？
- 在个人中心页面点击“提现”
- 输入提现金额
- 选择提现方式
- 点击“提交”按钮
- 提现申请将在3个工作日内处理

## 八、注意事项

### 8.1 安全提示
- 请确保您已取得合法的驾驶证和教练证
- 陪练过程中请遵守交通规则
- 如遇紧急情况，请立即采取措施并报警

### 8.2 教学规则
- 请按照约定时间和地点提供陪练服务
- 教学过程中请耐心指导学员
- 不得向学员收取额外费用

### 8.3 服务规范
- 保持良好的职业形象
- 尊重学员的隐私和权益
- 不得泄露学员的个人信息

### 8.4 系统使用
- 请及时更新个人信息
- 定期查看课程预约情况
- 如有系统问题，请及时联系管理员

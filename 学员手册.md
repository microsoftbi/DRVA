# 驾驶陪练系统学员手册

## 一、系统概述
欢迎使用驾驶陪练系统！本系统旨在帮助有驾驶证的人进行驾驶陪练服务，您可以通过本系统预约教练的陪练课程，提高驾驶技能。

## 二、注册与登录

### 2.1 注册账号
1. 打开驾驶陪练系统学员端页面
2. 点击“注册”按钮
3. 填写注册信息：
   - 姓名：请输入您的真实姓名
   - 手机号：请输入您的常用手机号
   - 邮箱：请输入您的常用邮箱（可选）
   - 密码：请设置6位以上的密码
   - 区域：请选择您所在的区域
4. 点击“注册”按钮完成注册

### 2.2 登录系统
1. 打开驾驶陪练系统学员端页面
2. 输入您的手机号和密码
3. 点击“登录”按钮
4. 登录成功后，您将进入学员首页

### 2.3 忘记密码
1. 在登录页面点击“忘记密码”
2. 输入您的手机号
3. 点击“获取验证码”
4. 输入收到的验证码
5. 设置新密码
6. 点击“确认”按钮完成密码重置

## 三、预约课程

### 3.1 查看可用课程
1. 登录系统后，进入学员首页
2. 您可以看到所有可用的陪练课程
3. 可以通过以下方式筛选课程：
   - 区域筛选：选择您所在的区域
   - 日期筛选：选择您想要预约的日期
   - 时段筛选：选择上午、下午或晚上

### 3.2 预约课程
1. 在课程列表中选择您感兴趣的课程
2. 点击课程卡片上的“预约”按钮
3. 确认预约信息
4. 点击“确认预约”按钮
5. 预约成功后，课程状态将变为“待确认”

### 3.3 取消预约
1. 点击底部导航栏的“我的课程”
2. 找到您想要取消的课程
3. 点击“取消预约”按钮
4. 确认取消操作
5. 取消成功后，课程状态将变为“已取消”

## 四、我的课程

### 4.1 查看课程列表
1. 点击底部导航栏的“我的课程”
2. 您可以看到所有已预约的课程
3. 课程状态说明：
   - 待确认：您已预约课程，等待教练确认
   - 已确认：教练已确认您的预约
   - 已取消：您已取消预约
   - 已完成：课程已结束

### 4.2 课程详情
1. 点击课程卡片进入课程详情页面
2. 查看课程的详细信息：
   - 教练姓名和联系方式
   - 课程日期和时段
   - 课程状态
   - 课程地点

## 五、个人中心

### 5.1 个人信息
1. 点击底部导航栏的“我的”
2. 查看您的个人信息：
   - 姓名
   - 手机号
   - 邮箱
   - 所在区域

### 5.2 修改个人信息
1. 在个人中心页面点击“修改信息”
2. 修改您的个人信息
3. 点击“保存”按钮

### 5.3 修改密码
1. 在个人中心页面点击“修改密码”
2. 输入原密码
3. 输入新密码
4. 确认新密码
5. 点击“保存”按钮

### 5.4 退出登录
1. 在个人中心页面点击“退出登录”
2. 确认退出操作
3. 系统将返回登录页面

## 六、常见问题

### 6.1 如何选择合适的教练？
- 可以查看教练的评分和评价
- 选择与您所在区域相同的教练
- 选择您方便的时间段

### 6.2 预约课程后可以取消吗？
- 可以在课程开始前取消预约
- 取消预约不会产生任何费用
- 请尽量提前取消，以便其他学员预约

### 6.3 教练确认课程后可以取消吗？
- 教练确认课程后，如需取消请提前24小时通知教练
- 如临时取消可能会影响您的信用评分

### 6.4 如何联系教练？
- 在课程详情页面可以查看教练的联系方式
- 可以通过系统内置的消息功能与教练沟通

### 6.5 课程结束后如何评价教练？
- 课程结束后，系统会自动提示您评价教练
- 您可以对教练的教学质量、态度等进行评分和评价
- 您的评价将帮助其他学员选择合适的教练

## 七、注意事项

### 7.1 安全提示
- 请确保您已取得合法的驾驶证
- 陪练过程中请遵守交通规则
- 如遇紧急情况，请立即联系教练或报警

### 7.2 预约规则
- 每个学员最多可以同时预约3门课程
- 请在课程开始前30分钟到达约定地点
- 如迟到超过15分钟，教练有权取消课程

### 7.3 费用说明
- 陪练费用请与教练协商确定
- 系统不收取任何中介费用
- 请在课程结束后直接支付给教练
